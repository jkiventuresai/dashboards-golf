Golf Dashboard

Public landing page for the live golf performance dashboard, score tracker reports, and historical dashboard versions.

Active Dashboard

The main dashboard is dashboard-v9.html. It uses the updated score data and the restored full 2026 dashboard layout with charts, insights, tables, and course performance views.

Open Active Dashboard View dashboard-v9.html in GitHub Open GitHub Repo
Current setup: dashboard-v9.html loads the external data file from data/rounds-v9.json, so the dashboard can be refreshed by updating the data file without rebuilding the whole page.

Dashboard Tracks

  • Score trends and rolling 5-round performance.
  • Tracker handicap trend.
  • Course performance and course map.
  • Temperature, seasonal, and play-frequency patterns.
  • Hole-by-hole scoring, 3-hole segment fatigue, and back-nine fade.
  • Course ratings, round detail table, and SWOT-style golf insights.
