Data Management Notes

This HTML file is a static embedded-data snapshot. Adding rows to a CSV will not update this file automatically.
Correct update workflow: add new round to CSV → save file → ask ChatGPT to regenerate next version, such as dashboard-golf-v7.html.
For the course map, the best next data upgrade is adding latitude and longitude columns to golf-course-locations-mapping.csv.
